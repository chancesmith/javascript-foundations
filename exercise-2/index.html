What are comments?

Software development is collaborative. We use comments to communicate how the code works. This helps others dive in to code they didn't write and reminds us when we return days later. Comments is a way to leave notes for other developers.

The computer ignores the comments when running the program.

How do you use comments?

Single line comments are denoted with two forward slashes //. 

              // Prints 10 to the console 
              console.log(10)

Multi-line comments are denoted with /* */ 

              /*
              Prints 10 to the console 
              console.log(10)
              */
