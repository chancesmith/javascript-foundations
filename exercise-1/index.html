What is a console?

The console shows information to the developer, like errors, that helps determine what the software has done.

The `console` is an object, a collection of data and actions, we can use in our code.

How do you use the console?

The `.log` is how we print a message we wish to send to the console.

Example

console.log('your message here', data)
